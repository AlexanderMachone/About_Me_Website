# 陶爷个人介绍网页

这是陶爷（王烨陶）的个人介绍静态网页，用于展示企业 AI 咨询落地、飞书产品搭建、业务架构陪跑等服务。

## 本地查看

直接用浏览器打开 `index.html` 即可。

也可以在本目录启动本地预览服务：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```
